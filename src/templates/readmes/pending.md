# pending/

This folder contains active epics and their stories.

## Structure

```
{{pendingDir}}/
  NN-epic-name/
    _epic.md          ← Epic overview (objectives, acceptance criteria, key decisions)
    story-NN-name.md  ← Individual story
    story-NN-name.md
  NN-another-epic/
    _epic.md
    story-NN-name.md
```

## Conventions

- Epic numbers are assigned sequentially at creation time. Working order is independent of creation order.
- Stories capture **what** the user needs and **why**, not how to build it.
- No story points. No estimation. No sprints.
- Stories are living documents — description, acceptance criteria, and key decisions can be refined as understanding sharpens.
- When work is complete, move the story (and epic, if fully done) to `{{doneDir}}/` using the `/alphaspec-complete-story` prompt.
