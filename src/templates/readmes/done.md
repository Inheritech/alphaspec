# done/

This folder contains completed epics and stories, archived for reference.

## Structure

```
{{doneDir}}/
  NN-epic-name/
    _epic.md          ← Epic overview (as it stood at completion)
    story-NN-name.md  ← Completed story (includes Implementation Notes section)
```

## What's here

Completed stories have an **Implementation Notes** section appended at the bottom. This is a brief technical record of what was built — the main modules involved, any non-obvious decisions made during implementation, and where to look if something needs to change.

## Why keep completed work?

The `{{doneDir}}/` folder is the project's long-term memory. It tells the AI assistant what the project has already been through, how things were decided, and where the edges are. When starting a new session, reviewing recent `{{doneDir}}/` entries helps the AI orient quickly.
